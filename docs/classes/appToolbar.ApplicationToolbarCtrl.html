    appToolbar.ApplicationToolbarCtrl Class

    Defined in: src/app/appToolbar/appToolbar.js:3
    Module: ozpWebtopApp

    Application toolbar on the bottom of Webtop. Contains a menu for adding favorited apps to the dashboard and shows names/icons for all apps currently on the dashboard. Clicking on the app name/icon buttons in the toolbar results in the application being shown or hidden from the dashboard view (in desktop layout)
